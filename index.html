JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.

Because it is one of the most widely used languages worldwide and its learning curve is one of the easiest compared to languages like Java or PHP that have similar performance, so it is also highly recommended for learning to program.
